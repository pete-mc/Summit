Topo Reporting App

Topo is a Blazor WebAssembly application designed to assist scout leaders. It enables efficient and easy generation of reports from the Scouts Australia Terrain system.

Key features of Topo include:

  • Interactive and user-friendly interface for data entry and report generation.
  • Seamless integration with Scouts Australia's Terrain system.
  • Real-time data processing and visualization tools.
  • Accessible as a web application without the need for installation.
  • Comprehensive documentation and support via its Wiki page.

Please click the button to access Topo. A new tab will open where Topo will start. We are hoping to remove the need to login again in a future release.
